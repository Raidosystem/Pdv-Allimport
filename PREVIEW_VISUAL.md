```
   - From: secondary-900

   - Overlay: primary-500/10 blur
```

## 🎬 ANIMAÇÕES

### **1. Entrada dos Cards**
```
Stagger animation (aparecem um por um)
Delay: 50ms entre cada card
Fade in + Slide up
Duration: 300ms
```

### **2. Transição Seleção → Senha**
```
Fade out cards (200ms)
Slide in formulário senha (300ms)
Focus automático no input
```

### **3. Loading States**

#### Carregando usuários:
```
╔════════════════════════╗
║                        ║
║    ⟳ (spinner)         ║
║                        ║
║  Carregando usuários...║
╚════════════════════════╝
```

#### Validando senha:
```
┌────────────────────────┐
│  ⟳ Entrando...         │  ← Botão com spinner
└────────────────────────┘
```

### **4. Feedback Visual**

#### Sucesso:
```
toast.success('✅ Usuário criado com sucesso!')
  ↓ (aparece no canto superior direito)
┌─────────────────────────────────────┐
│ ✅ Usuário criado com sucesso!      │
└─────────────────────────────────────┘
     (desaparece após 3s)
```

#### Erro:
```

┌─────────────────────────────────────┐
│ ❌ Senha incorreta                  │
└─────────────────────────────────────┘

```

```
- outline: 2px solid #3b82f6

- Visible focus ring
```

### **Keyboard Navigation**
```
Tab       → Próximo elemento
```
