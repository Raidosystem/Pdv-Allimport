Agora você pode encontrá-lo em:

  • 📍 Menu Iniciar → Procure por "PDV Allimport"
  • 🖥️ Área de trabalho (se criou atalho)
  • 📌 Barra de tarefas (clique direito para fixar)

🆘 Não Encontra a Opção?

Possíveis soluções:

  • ⏰ Aguarde alguns segundos após carregar a página
  • 🔄 Atualize a página (F5 ou Ctrl+R)
  • 🔐 Faça login no sistema (pode ser necessário estar logado)
  • 🌐 Verifique o HTTPS (URL deve começar com https://)
  • 🔄 Teste outro navegador (Chrome geralmente funciona melhor)
